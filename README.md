## Feature

| Plugin                     | Desc                                             |
| -------------------------- | ------------------------------------------------ |
| chrisgrieser/nvim-scissors | Automagical editing and creation of snippets.    |
| pysan3/fcitx5.nvim         | switch and restore fcitx state for each buffer.  |
